Alur Sistem

  1. Pelanggan membuka katalog obat.
  2. Sistem mengecek apakah katalog sudah tersimpan sementara.
  3. Jika belum ada, data diambil dari database utama.
  4. Data katalog disimpan sementara untuk mempercepat akses berikutnya.
  5. Jika admin mengubah stok, data sementara dibersihkan agar stok tetap akurat.
  6. File statis seperti CSS, JavaScript, dan gambar dapat disimpan melalui cache/CDN.
